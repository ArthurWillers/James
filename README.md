# James

O James é um software de assistência para professores, projetado como uma ferramenta 100% livre focada em automatizar a correção de provas. O seu principal objetivo é eliminar o trabalho manual e repetitivo, permitindo aos professores criar avaliações, definir gabaritos e corrigir automaticamente os cartões-resposta dos alunos através do processamento de imagens (OMR) de fotos ou digitalizações.
